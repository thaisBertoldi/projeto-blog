Projeto desenvolvido no curso Formação Node.js com o intuito de criar uma aplicação que permita a criação, leitura, exclusão e alteração (CRUD) de categorias, artigos e
usuários, com as tecnologias: 
-Javascript
-Express
-EJS
-Sequelize
-Slugify
-Bcryptjs
-Mysql2
-Body-parser
-Express-session
-Bootstrap
-Tinymce
